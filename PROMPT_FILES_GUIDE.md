## 📂 实际项目结构

当前项目采用扁平化结构，核心文件在根目录，数据管理在 `database/` 和 `chapters/` 目录：

```
vibe-noveling/
├── rules.md                    # ✅ 核心规则
├── timeline.md                 # ✅ 时间线
├── outline.md                  # ✅ 大纲
├── setting.md                  # ✅ 世界观
├── characters.md               # ✅ 角色设定
├── context.md                  # ✅ 沟通上下文
├── database.md                 # ✅ 数据库说明
├── checklist.md                # ✅ 检查清单
├── style_guide.md              # ✅ 风格指南
├── PROMPT_FILES_GUIDE.md       # 📋 本文档（扩展指南）
├── chapters/                   # ✅ 章节目录
│   └── README.md
└── database/                   # ✅ 数据库目录
    ├── database_template.md    # ✅ 数据库模板
    ├── README.md               # ✅ 数据库说明
    └── list/                   # ✅ 列表文件目录
        ├── items_list.md       # ✅ 物品追踪
        ├── skills_list.md      # ✅ 技能追踪
        ├── relationships_list.md # ✅ 关系追踪
        └── foreshadowing_list.md # ✅ 伏笔追踪
```
